# ChatOS 3D Realistic Room Prototype

这是与现有 ChatOS 前端完全隔离的视觉原型，不会修改或依赖原来的聊天页面代码。

## 运行

```bash
npm install
npm run dev
```

默认访问：<http://127.0.0.1:4388>

## 原型范围

- 写实风格的程序化 3D 房间和真实窗景
- 书桌、电脑、手机、左墙实体木质项目书架（每页 6 个项目）和右侧实时任务大屏
- 电脑内置登录锁屏；登录后进入无内边框全屏桌面
- 电脑桌面上的 AI 聊天、终端和远程连接三个全屏交互应用
- AI 聊天沿用原前端的数据关系：先添加 Agent 联系人；项目聊天从项目负责人列表进入；首次发送消息时才建立对应会话
- 聊天支持完整历史、加载更多、Markdown/代码复制、多行输入、附件、模型和思考等级、Reasoning、Plan Mode、停止生成
- 桌面动态小猫
- 白天、黄昏、夜晚切换
- 可推进观察点的模拟聊天、终端、远程连接、实体项目档案和任务墙

## 真实数据

电脑登录页已经接入 ChatOS 的联系人、Agent、项目负责人、会话、消息、任务、模型配置和 WebSocket 实时事件。默认 API 地址：

```text
http://127.0.0.1:3997/api
```

可通过 `VITE_CHATOS_API_BASE_URL` 覆盖。后端未启动时，可以在电脑登录页选择“先进入演示桌面”，不会影响房间预览。

## 性能策略

- 3D 场景使用按需渲染，镜头和光照静止后停止连续绘制
- 进入电脑全屏后隐藏并暂停房间层，后台标签页暂停 WebSocket
- 图片附件发送前按需压缩，附件总大小限制为 20 MB
- 生产构建拆分 React、图标和 Three.js 运行时，便于缓存和并行加载
