# Showcase Images

This directory stores website showcase screenshots captured from local Chatos RS services.

Recommended filenames:

- `chatos-main.png`
- `memory-engine.png`
- `task-runner.png`
- `sandbox-manager.png`
- `project-management.png`

Before committing or publishing screenshots, remove tokens, private paths, usernames, project secrets, and unrelated local data.
