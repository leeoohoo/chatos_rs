这个 MCP 用于为当前项目搜索、复用或创建沙箱镜像。

使用规则：
1. 每次初始化都先调用搜索工具获取当前真实镜像状态，不要复用 Memory 或历史记录中的旧结论。
2. 优先复用满足语言、运行时与依赖服务要求的可用镜像；没有合适镜像且项目可运行时再创建。
3. 创建镜像时准确提供 features、端口、环境变量和必要的非交互构建脚本。脚本必须可重复执行、使用 `set -e`、不得包含密钥。
4. 创建操作应同步等待最终结果；成功后只使用工具真实返回的 `image_id`、`image_ref` 和状态，不要自行构造。
5. 失败时保留原始错误并停止声称环境已经就绪，避免无依据地重复创建相同镜像。
