# ChatOS Presentations

Use this Skill to inspect or create PPTX presentations in the authorized local workspace.

- Use `inspect_pptx` to validate an existing package and count its slide files.
- Use `create_pptx` to create a basic widescreen deck from ordered title/body slide definitions.
- Keep each slide focused. Prefer a short title and concise body text; split dense material across multiple slides.
- The current native writer provides editable text-only slides with a stable built-in theme. Images, charts, speaker notes, complex layouts, animations, and render-based visual QA are not exposed yet.
- Never claim that PowerPoint or a cloud presentation service was controlled; the output is generated by the Local Connector adapter.
